page 지시자
- "ex03.jsp"

- JSP 페이지 실행과 관련된 여러가지 정보를 톰캣에게 알려주는 역할
- JSP 페이지의 첫줄에 작성한다.
- page 지시자를 여러개 작성 가능

<%@page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>
	a. language="java"
		- JSP 페이지에서 사용할 프로그래밍 언어 지정
	
	b. contentType="text/html; charset=UTF-8"
		- 동적으로 만들어진 웹 페이지에 대한 설정
		- text/html: MIME 타입 > 돌려준 데이터 HTML 문서라는 것을 브라우저에게 알려주는 역할
		- charset=UTF-8: 돌려준 데이터 HTML 문서의 인코딩
	
	c. pageEncoding="UTF-8"
		- 현재 JSP 파일의 인코딩

내장 객체의 생명 주기
- pageContext, request, session, application
- 데이터 저장 가능 객체

- pageContext: 생성(페이지 요청) ~ 소멸(페이지 반환)
- request    : 생성(페이지 요청) ~ 소멸(페이지 반환)
			   생성(페이지 요청) ~ 페이지 이동(forward()) ~ 소멸(다음페이지반환)
- session	 : 생성(사용자가 사이트에 처음 방문) ~ 소멸(사용자가 사이트에서 나갈 때)
- application: 생성(사이트에 첫번째 사용자가 방문) ~ 소멸(사이트에 마지막 사용자가 나갈 때)
